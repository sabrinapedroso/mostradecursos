Desenvolvimento Web

Sobre

Em desenvolvimento web, os alunos são introduzidos aos conceitos de internet (o que é, como funciona), relações backend e frontend e, principalemente, o funcionamento e desenvolvimento de páginas da web.
São trabalhadas as linguagens de programação html, css e javascript. Além disso, a disciplina contempla a utilização de softwares de versionamento como o GitHub, onde os alunos são incentivados a publicar seus projetos.

Mostra de cursos 2022

Este site foi desenvolvido pela aluna Sabrina A. Pedroso como forma de divulgar o curso de informática na mostra de cursos 2022, aplicando seus conhecimentos de desenvolvimento web
